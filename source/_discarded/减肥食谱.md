针对年后减肥，个人定制，供参考
预计每天摄入能量：1600kcal ～ 1700kcal，时间一个月，预计瘦八斤

早上：300ml 脱脂牛奶 + 代餐粥
能量：34kcal * 3 + 115kcal =217 kcal

午餐：120g 荞麦面 + 三个鸡蛋 + 腊肠/腊肉/烤肉 （我把午餐作为脂肪的主要摄入路径）
能量：400kcal + 约200kcal + 约400kcal = 1000kcal

晚餐：300ml 脱脂牛奶 + 代餐粥
能量：34kkcal * 3 + 115kcal = 217kcal

零食：牛肉干、脱脂牛奶

补剂：swisse 多维片，钾片

调味品：老干妈、椒盐粉、辣椒粉、香油、酱油、黑芝麻酱
